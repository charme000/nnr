设置回调处理函数

参数:
ConnectServerSuccess成功连接到服务端时触发
AcceptNewClient成功接受到客户端连接时触发
ConnectError连接过程中发生错误时触发
ServerDisconnected服务端退出时触发
ClientDisconnected任一客户端退出时触发
AddToBusFailed添加用户实体到Bus中时如果对应实体已经存在时触发
DataArrived有数据到达时触发
ByteWriten主动发送数据写入完毕时触发
警告:
以上参数如果不需要处理相应的动作,则可以设置为NULL.以上动作都为异步动作.
